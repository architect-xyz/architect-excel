# architect-excel
Excel plugin to access Architect backend via the Typescript API
https://github.com/architect-xyz/architect-ts



## To Run

```bash
npx webpack
```

The `src/architect-excel.ts` is the main file
This gets compiled into `public/taskpane.js`
