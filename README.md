# architect-excel
Excel plugin to access Architect backend via the Typescript API
https://github.com/architect-xyz/architect-ts



## To Run


```bash
npm install
npx webpack
```

### Adding Functions

One must add functions to:
- manifest.xml
- architect-excel.ts
- functions.json
