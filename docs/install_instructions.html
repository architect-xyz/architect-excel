architect-excel Install Instructions

Follow these steps to install the architect-excel add-in in Excel:

  1. Open Excel and navigate to the Insert tab.
  2. Click on Office Add-ins and then select Manage My Add-ins.
  3. Choose Upload My Add-in.
  4. Locate and select the manifest.xml file.
  5. Once uploaded, the add-in will appear in Excelâ€™s Task Pane and is ready to use.
